What is the purpose of the motor driver/shield?

  The purpose of the motor driver/shield is to connect the Arduino to the motors and allow the code run by the Arduino to run the motors. This also stops the motors from directly having access to the Arduino and provides places to solder header pins, making the whole setup more secure.
